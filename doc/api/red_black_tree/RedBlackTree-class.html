A Linked-List Red-Black tree implementation.

Insert and delete operations automatically update an item's position in the linked list, resetting next and prev pointers accordingly. When ordering items in the tree, the item's < > and == operators are used.

Because this is a red-black tree, it is not guaranteed to be completely balanced after write operations.
